## Linux commands learned :-

- **file permission in linux**
  - `chmod` : for modifying permissions of particular directories or files.
  - `chown` : to modify the owner of file or directories to user or a group.
  - creating and adding user to groups.
  - switching user

- **types of users:-**
  - **root user**: superuser with full control over system.
  - **regular user** : non admin user can be created also.
  - **system user** : user created by system.
