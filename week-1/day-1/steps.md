## Installing VS Code and GIT:-

- Download and install VS code in system.
- Download and install GIT in your system.
- Use the default options and install the VS Code.
- Use the default options and install the GIT.

## Steps to work with GIT using vs code:-

- Open VS Code and a folder where we will work.
- Now open terminal using ctrl + ~.
- Now in opened terminal press the dropdown beside the + icon in terminal.
- Now select "Git Bash" option for a new terminal.
- In this new terminal window can do the Git commands.

## Try Using :-

- `git --version` (or) `git -v`

> If the command does not shows error the git is installed and working.
