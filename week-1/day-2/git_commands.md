## Git Commands:-

- `git add .` : To add all files and folders into staging area. Will be added in next commit.
- `git commit -m "can write a msg"` : To commit all the files and folders present in staging area.
- `git push origin <branch>` : To push the changes to remote repo for particular branch.

> **If the origin is not specified (First time push):-**
- `git remote add origin "<remote repo url>"` : To add the remote repo path for push/pull.
- `git remote -v` : To check if the remote repo is added.
- Then push the commit to remote repo.
